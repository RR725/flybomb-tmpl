一个react脚手架，集成周边生态，基于webpack构建，各模块按需加载。

# 库
- react
- react-router
- redux
- axios
- immutable
- mockjs
- jest
- normalize.css

# 框架
- antd

# 兼容性
- IE9+
- Chrome
- Firefox
- Safari

# 开发调试
```
npm i
npm run start-dev
```
